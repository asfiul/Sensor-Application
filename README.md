# Sensor-Application
This is an demo application of sensor android application. Four type of sensor implemented this Application.
Using Cardview in XML(Main-Activity) for showing sensor value. 
##Call Four type of sensor(Light sensor,proximity sensor, accelerometer, and Gyroscope sensor.) in java(Main-activity)
Using Foreground notification mathod for runing the service in the background, so even if the user closes it from the task manager,
the app keeps running in the background
